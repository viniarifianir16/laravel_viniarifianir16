# Hospital App
## Start Project
```
git clone https://github.com/viniarifianir16/laravel_viniarifianir16.git
composer install
composer run dev
```

## Version
```
Laravel 12
PHP 8.2
npm 11.5.2
Node 22.19.0
```
    
## Features
- CRUD Hospital Data
- CRUD Patient
- Filter Data by Hospital
- Support Dark Mode
- Responsive Mobile

## Library
- Livewire Starter Kit
- Laravel Volt
- Flux UI
- TailwindCSS
- Bootstrap
- DataTables
- Sweetalet2
- Bootstrap Icons
- Heroicons
- Select2 jQuery
